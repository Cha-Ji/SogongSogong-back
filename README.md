### Main Merge 방식
- 각자의 Branch를 만들어 코드를 작성한다.
- 코드 작성이 완료되면 각자 브랜치로 Commit & Push를 한다.
- 이후 본인의 브랜치를 Main으로 Pull Request를 추가한 뒤 설명을 한다.
- 각자 Pull Request에 대한 리뷰를 남긴 후 Merge를 승인한다.
